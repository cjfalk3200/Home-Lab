## Overview
The goal of this project is to create a small Homelab environment as the basis for future projects exploring different concepts in cybersecurity. All VMs will be hosted using VM Worstation Pro and configured for each project. I used the following repository CyberSec-Xperience (https://github.com/rashidaalexander/CyberSec-Xperience?) as the steps to set up this lab.This environment will allow for practicing on penetration testing, network defense, incident response, and security monitoring.
